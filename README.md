# Clone Bitbucket Repositories

This script clones repositories from Bitbucket based on a search term.

## Usage

```bash
pnpm dev
```

## Configuration

```bash
CLONE_PATH=~/code pnpm dev
```

```bash
cp .env.example .env
```

## Environment Variables

```bash
BITBUCKET_USERNAME=
BITBUCKET_PASSWORD=
CLONE_PATH=~/code
```
